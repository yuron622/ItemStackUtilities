ItemStackUtilities is a class extends ItemStack.
You'll be able to change display names or lore of any ItemStack objects smoothly.

Here's the list of the methods contained:

  ItemStack getSetDisplayName(Component):
    returns ItemStackUtilities object with a new display name you specified. So you can get a new ItemStackUtilities by writing like "itemStackUtilities.getSetDisplayName(component)"
    
  ItemStack getAddLore(Component):
    returns ItemStackUtilities object, adding a new lore you specified. So you can get a new ItemStackUtilities by writing like "itemStackUtilities.getAddLore(component)"
