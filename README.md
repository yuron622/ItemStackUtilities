ItemStackUtilities is a class extends ItemStack.
You'll be able to change display names or lore of any ItemStack objects smoothly.

Here's the list of the methods contained:

  ItemStack getSetDisplayName(Component):
    returns ItemStack object with a new display name you specified. So you can get a new ItemStack by writing like "itemStack.getSetDisplayName(component)"
    
  ItemStack getAddLore(Component):
    returns ItemStack object, adding a new lore you specified. So you can get a new ItemStack by writing like "itemStack.getAddLore(component)"
